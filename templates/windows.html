Windows setup

Install into the current user's Trusted Root Certification Authorities store.

Download CertificateDownload PowerShell Helper

  1. Download the certificate.
  2. Double-click the file.
  3. Select Install Certificate.
  4. Choose Current User.
  5. Select Place all certificates in the following store.
  6. Choose Trusted Root Certification Authorities.
  7. Finish and accept the security warning.

PowerShell helper

powershell -ExecutionPolicy Bypass -File .\windows.ps1
