Windows setup

Use the Windows installer for the fastest path. It downloads this portal's PowerShell helper, asks for administrator approval, verifies the certificate fingerprint, imports the certificate into the Local Computer Trusted Root store, and opens the verification page.

Download Windows Installer Download PowerShell Helper Download Certificate Only

Installer

  1. Download the Windows installer.
  2. Open the downloaded windows.cmd file.
  3. Approve the administrator prompt.

PowerShell helper

powershell -ExecutionPolicy Bypass -File .\windows.ps1

Manual install

  1. Download the certificate.
  2. Double-click the file.
  3. Select Install Certificate.
  4. Choose Local Machine.
  5. Select Place all certificates in the following store.
  6. Choose Trusted Root Certification Authorities.
  7. Finish and accept the security warning.
