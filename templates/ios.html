iPhone / iPad setup

iOS and iPadOS require manual approval. A website cannot silently install and trust a root certificate.
Download Configuration Profile Download Certificate Only
  1. Download the configuration profile.
  2. Open Settings.
  3. Tap Profile Downloaded.
  4. Install the profile.
  5. Go to Settings → General → About → Certificate Trust Settings.
  6. Enable full trust for the {{ cert_display_name }} certificate.
  7. Restart Safari or your browser.
