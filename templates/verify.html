Verify setup

After installing the certificate, close and reopen your browser, then visit the test HTTPS site. If the test site still shows a certificate warning, the install did not complete successfully for this browser or device.

Open Test HTTPS Site

Success looks like

  • The page loads without a certificate warning.
  • The browser lock icon shows a valid connection.
  • The certificate chain shows the company/Cato inspection certificate as the issuer.

If it still fails

  • Run the installer again from the setup page.
  • Restart the browser.
  • Check whether Firefox needs a separate import.
  • Make sure the certificate was trusted as a root, not merely downloaded.
  • Contact {{ support_email }}.
