macOS setup

macOS requires administrator approval before trusting a root certificate.
Download macOS Helper Download Certificate Only

Helper install

chmod +x macos.sh
sudo ./macos.sh

Manual install

  1. Download the certificate.
  2. Open Keychain Access.
  3. Select the System keychain.
  4. Import the certificate.
  5. Open it, expand Trust, and set it to Always Trust.
  6. Restart your browser.
